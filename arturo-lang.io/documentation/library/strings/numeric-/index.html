numeric?

check if given string is numeric

Parameters

numeric? string :char :string

Returns

  • :logical

Examples

copy
numeric? "hello"           ; => false
numeric? "3.14"            ; => true
numeric? "18966"           ; => true
numeric? "123xxy"          ; => false
